# Forum-de-Perguntas-NodeJS
Projeto NodeJS para a criação de um fórum de perguntas. 

#Tecnologia
NodeJS
Express
EJS

#Demais infos
Utilizado Sequelize para conexão e manipução do banco de dados.
Bootstrap para estilização
EJS para renderizar HTML
